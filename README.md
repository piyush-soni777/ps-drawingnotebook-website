# 🎨 Drawing Notebook

Drawing Notebook is an interactive web-based drawing application built using HTML, CSS, and JavaScript.  
It allows users to draw freely on a digital canvas using a pencil tool, just like a real notebook.

---

## 🚀 Live Demo
🔗 https://ps-drawingnotebook-website.netlify.app

---

## ✨ Features
- ✏️ Pencil tool for freehand drawing  
- 🧾 Clean white canvas (notebook style)  
- 🎨 Smooth drawing experience  
- 🖱️ Mouse-based drawing controls  
- 📱 Responsive design  

---

## 🛠️ Technologies Used
- HTML5 (Canvas)  
- CSS3  
- JavaScript  

---

## ⚙️ How It Works
- The canvas element is used as a drawing board  
- Mouse events are captured using JavaScript  
- Lines are drawn dynamically as the user moves the cursor
